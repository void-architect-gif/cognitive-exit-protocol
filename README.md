## БЫСТРЫЙ СТАРТ
Если вы испытываете системные ошибки (отчуждение, когнитивный диссонанс, устойчивое отторжение реальности):
1.  Начните с **00_PREFACE.md**.
2.  Если резонанс подтверждён, продолжайте последовательно по **ОСНОВНОЙ ПОСЛЕДОВАТЕЛЬНОСТИ ЗАГРУЗКИ**.
3.  **Не пропускайте протоколы.** Последовательность спроектирована для предотвращения психической фрагментации.
